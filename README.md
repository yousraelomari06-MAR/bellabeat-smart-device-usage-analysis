# Bellabeat Smart Device Usage Analysis
## Dashboard Preview
![Bellabeat Dashboard](Bellabeat%20dashboard.png)
## Project Overview
This project is part of the Google Data Analytics Professional Certificate capstone.
The goal of the analysis is to explore smart device usage patterns and generate data-driven insights to support Bellabeat's marketing and product strategy.

Using Fitbit public data as a proxy for Bellabeat users, this analysis identifies behavioral trends in daily activity and highlights opportunities to imrpove user engagement and retention.

---
## Business Task
Analyze user activity data to understand engagement behavior and identify opportunities for improving long-term smart device usage.

---
## Dataset 
The dataset used for this project is the **Fitbit Fitness Tracker Data**, publicly available on Kaggle.

It includes daily user activity metrics such as :
- Steps
- Calories burned
- Activity intensity
- Distance travelled
Data Source:
https://www.kaggle.com/datasets/arashnic/fitbit

---
## Tools Used
- **SQL (Google BigQuery)** - data cleaning and exploration
- **Tableau Public** - data visualization and dashboard creation

---
## Key Insights
- User engagement is inconsistent across the observed period.
- Several users recorded days with **zero steps** indicating inactivity or device disengagement.
- Activity levels vary significantly across weekdays, suggesting that daily routines influence device usage.

---
## Recommendations
- Implement **personalized reminders** to re-engage inactive users.
- Provide **behavioral insights based on weekday activity trends**.
- Improve onboarding to help users integrate the device into their daily routines.
